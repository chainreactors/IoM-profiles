# IoM Profiles

本仓库包含多个配置模板，用于展示 malefic 的不同协议、模块配置以及 prelude 功能的使用方式。

## Malefic 协议配置模板

- **malefic-http** - 基于 HTTP 协议的 malefic 配置模板
- **malefic-https** - 基于 HTTPS 协议的 malefic 配置模板
- **malefic-tcp** - 基于 TCP 协议的 malefic 配置模板
- **malefic-tcp-tls** - 基于 TCP+TLS 的 malefic 配置模板
- **malefic-rem** - 使用 REM（Remote Execution Module）的 malefic 配置模板

## Malefic 模块配置模板

- **malefic-nano-modules** - 最小化modules（禁用热加载）
- **malefic-mini-modules** - 支持热插拔的最小化配置（开启热加载）

## Prelude 功能示例

- **prelude-persist** - prelude 持久化功能示例（包含 BOF 持久化和 shellcode 执行）
- **prelude-shellcode** - prelude shellcode 执行功能示例

## Malefic 与 Prelude 集成

- **malefic-with-prelude** - malefic 搭配 prelude 使用的配置模板
